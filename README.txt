Codigo relativo ao pacman encontra-se dentro da directorio pacman. 
Para correr o aplicativo, terá de executar o seguinte comando:

 	->python pacman.py -p NeuralNetworkAgent  -g iiaGhostAgent --ghostArgs fnStrategy='crazystrategy'
	
Este comando irá correr o jogo no mapa default , e com fantasmas com a estratégia crazystrategy (agente reactivo criado no TP1 , modificado)

Os ficheiro de treinos desenvolvidos que se apresentam dentro da pasta pacman são os desenvolvidos pelo jogador humano v2.0.

Em Anexo:
	-> teste_humano
	-> teste_humanos_v2.0
	-> teste_reactivo

Estes directorios são os treinos desenvolvidos, com o agente reactivo e
agente Humano, como o nome indica.
